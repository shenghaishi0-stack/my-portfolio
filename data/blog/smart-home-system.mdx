## 项目概述

本项目是一套基于 **ESP32 系列芯片** + **Home Assistant** 平台深度二次开发的智能家居控制系统，融合三维可视化 UI、AI 大模型语音助手、多模态身份识别等前沿技术，打造出一个开放兼容、高度个性化的智能家居解决方案。

> 荣获**青少年科技创新大赛**奖项，系统已稳定运行，具备完整的商业落地能力。

## 系统架构

本系统采用**三层架构**设计：

```
┌─────────────────────────────────────────────────────────┐
│                      应用层                              │
│   Three.js 3D UI  |  Node-RED  |  AI 语音助手  |  Web端  │
└─────────────────────────────────────────────────────────┘
                            │ MQTT / WebSocket
┌─────────────────────────────────────────────────────────┐
│                      通信层                              │
│          Home Assistant  |  MCP Server  |  MQTT Broker  │
└─────────────────────────────────────────────────────────┘
                            │ WiFi / BLE
┌─────────────────────────────────────────────────────────┐
│                      感知层                              │
│   ESP32-S3  |  ESP32-CAM  |  AS608指纹  |  温湿度传感器  │
└─────────────────────────────────────────────────────────┘
```

## 功能模块详解

### 1. 三维可视化控制界面

![UI界面展示](/static/images/smart-home-ui.png)

基于 **Three.js** 技术，将用户实际住宅的 3D 模型导入系统，支持：

- 在 3D 模型中直接点击控制对应设备
- 实时显示设备在线状态（灯光、空调、门锁等）
- 支持 YAML 格式高级自定义配置
- 模块化卡片式设计，适配手机和电脑

### 2. AI 语音助手（本地部署）

![语音助手](/static/images/smart-home-voice.png)

在 **ESP32-S3** 上部署轻量级语音识别模型，实现：

- **本地唤醒** → 无网络延迟，即时响应
- **本地识别** → 语音数据不上云，保护隐私
- **AI 指令理解** → 结合 MCP Server 控制家居设备
- 支持对话式交互：「打开卧室灯」「空调调到26度」

> 相比 Siri / Google Assistant 依赖国外云端，本方案完全国产化部署，避免技术封锁风险。

### 3. 人脸识别门禁

使用 **ESP32-CAM** 采集图像，本地服务器通过 **OpenCV** 完成人脸识别：

- 识别成功 → MQTT 推送 → 自动开锁
- 生成出入日志，支持远程查看
- 适用于出租房、公寓、家庭门禁

### 4. 指纹识别

![硬件模块](/static/images/smart-home-hardware.jpg)

**AS608 指纹模块** 通过串口连接 ESP32：

- 本地比对，毫秒级响应
- 支持多指纹录入（家庭成员管理）
- 联动门锁执行开锁操作

### 5. Node-RED 低代码自动化

内置可视化流程编辑器，无需编程即可配置：

```
场景示例：
🌡️ 室温 > 30°C  →  自动开启空调
💡 识别到人脸  →  开灯 + 开锁 + 发送通知
🌙 22:00 准时  →  关闭所有非必要电器
🚪 离家模式    →  安防布防 + 节能模式
```

### 6. 多品牌设备兼容

通过 **MCP Server 协议转换 + MQTT 适配**，统一接入：

- 米家生态（小米全系）
- 涂鸦智能
- Aqara 绿米
- 普通 ESP8266/ESP32 自制设备

## 项目总结

本项目综合运用了**嵌入式开发、物联网通信、人工智能、三维前端可视化**等多个技术领域，实现了一套完整的、可商业落地的智能家居控制系统。

**技术收获：**

- 掌握 ESP32-S3 系列全栈开发（ESP-IDF + FreeRTOS + MQTT）
- 实践了 Home Assistant 开源平台的深度二次开发
- 学会了将 AI 大模型与嵌入式设备融合的 MCP 架构
- 积累了三维前端 Three.js 与物联网系统的集成经验

**未来规划：**

- 接入更多传感器（可燃气体、PM2.5）
- 优化 AI 语音模型，提升识别准确率
- 开发配套手机 App（UniApp 跨平台）
- 推进智慧社区、智慧养老场景落地
